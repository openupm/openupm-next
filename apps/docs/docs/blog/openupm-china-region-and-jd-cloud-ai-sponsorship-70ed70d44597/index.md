# OpenUPM China Region and JD Cloud & AI Sponsorship

<BlogPostMeta />

UPDATE: The China region has been discontinued.


OpenUPM is excited to announce that the [OpenUPM China region](https://openupm.cn/) is available to use today. The region is sponsored by [JD Cloud & AI](https://www.jdcloud.com/en), a leading cloud computing & AI platform in China.

![Image 6](./images/openupm-china-region-and-jd-cloud-ai-sponsorship-70ed70d44597-01-image-6.png)

Heart icon made by [Kiranshastry](https://www.flaticon.com/authors/kiranshastry) from [www.flaticon.com](https://www.flaticon.com/)

The China region aims to provide a better experience for visitors from China mainland:

*   The website and documentation translations.
*   Localized package metadata (title and description).
*   The mirror registry server one-way synced with the global region every hour at [https://package.openupm.cn](https://package.openupm.cn/)
*   The openupm-cli provides a dedicated `openupm-cn` command for the China region since version 1.14.0.
*   The report malware or abuse feature.
*   The Chinese blog at [Zhihu Column](https://www.zhihu.com/column/c_1308910319477874688).

JD Cloud & AI is very kind to sponsoring the infrastructure for the OpenUPM China region running on their platform. When talking to them earlier this year, they immediately recognize our value and love to help the open-source service to grow. We’re appreciated for the help from the contact Mr. Jiaxin Yao who helps arrange the sponsorship detail.

Thank you, JD Cloud & AI!

Please visit the OpenUPM China region at [https://openupm.cn](https://openupm.cn/).

For more sponsorship opportunities please visit [/support/#donations](/support/#donations).
